# Política de Segurança

## Versões Suportadas

Atualmente, o projeto está em desenvolvimento ativo.
Apenas a versão mais recente disponível no repositório é considerada suportada.

| Versão | Suporte |
| ------ | ------- |
| main   | ✅       |
| outras | ❌       |

---

## Reportando uma Vulnerabilidade

Se você identificar uma vulnerabilidade de segurança, siga as orientações abaixo:

* Não divulgue publicamente a vulnerabilidade antes da análise
* Abra uma issue no repositório com a tag **security**, ou entre em contato direto com os responsáveis pelo projeto

### O que incluir no reporte:

* Descrição detalhada do problema
* Passos para reproduzir
* Possível impacto
* Sugestão de correção (se houver)

---

## Processo

* O reporte será analisado pelos mantenedores do projeto
* Caso confirmado, será corrigido na versão mais recente
* O autor do reporte poderá receber retorno sobre o status da análise

---

## Observação

Este é um projeto acadêmico em desenvolvimento.
As práticas de segurança podem evoluir conforme o sistema cresce.
