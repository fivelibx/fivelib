# Product Backlog - FiveLib

## Épico 1: Gestão de Identidade e Acesso (Prioridade: Alta)
* **US01:** Como visitante, quero poder me cadastrar com nome, e-mail e senha, sendo maior de 18 anos, para ter acesso às áreas personalizadas. *(Ref: RF05, RN01, RN02)*
* **US02:** Como usuário registrado, quero fazer login na plataforma para acessar minha biblioteca pessoal. *(Ref: RF05)*

## Épico 2: Motor de Busca e Consulta (Prioridade: Alta)
* **US03:** Como visitante, quero buscar bibliotecas e ferramentas por nome, linguagem ou categoria na busca global. *(Ref: RF01)*
* **US04:** Como desenvolvedor, preciso garantir que a resposta da API na busca não exceda o limite de tempo estabelecido. *(Ref: RNF04)*

## Épico 3: Biblioteca Pessoal (Prioridade: Média)
* **US05:** Como usuário registrado, quero favoritar links do motor de busca para que fiquem salvos no meu perfil. *(Ref: RF02, RN03)*
* **US06:** Como usuário registrado, quero adicionar URLs externas privadas que só eu posso visualizar. *(Ref: RF04, RN03)*

## Épico 4: Suporte e Moderação (Prioridade: Baixa)
* **US07:** Como usuário, quero preencher um formulário de suporte reportando links quebrados. *(Ref: RF03)*
* **US08:** Como administrador, quero visualizar um painel com tickets de suporte com status "Pendente" para analisar e moderar. *(Ref: RN04, UC03, UC04)*
